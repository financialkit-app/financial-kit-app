# financial-kit-app
India’s Family Financial Manager – Manage documents, investments &amp; reminders in one app
🏦 Financial Kit – Family Financial Manager

Financial Kit is a mobile app designed for Indian families to manage all financial documents, investments, and reminders in one place.

## 🇮🇳 Problem

In most Indian families, important documents like LIC policies, insurance, FD receipts, and vehicle papers are scattered.
People forget premium dates, document expiry, and maturity dates.

## 💡 Solution

Financial Kit provides a single platform to:

Store all financial documents
Manage family members’ financial data
Track expiry dates and reminders
View complete financial summary
🚀 Key Features
👨‍👩‍👧 Family Kit (manage all members)
📄 Document storage (LIC, FD, Insurance, RC, etc.)
🔔 Smart reminders (premium, expiry, maturity)
📊 Dashboard (total investments overview)
🌐 Hindi & English support
📷 OCR document scanner (future feature)
📱 App Flow
Login (OTP / Gmail)
Select language (Hindi / English)
Dashboard
Add Document
Family Members
Reminder Alerts

Indian middle-class families
Tier 2 / Tier 3 cities
People managing multiple financial documents
